title: "getName"
description: "Queries the name of an attachment in Optimizely Campaign."
---

**Type**: String

**Parameters**

| Name         | Type   | Value                |
| ------------ | ------ | -------------------- |
| sessionId    | String | ID of the session    |
| attachmentId | long   | ID of the attachment |

**Return values**

Name of the attachment

**Code structure**

`String getName(string sessionId, long attachmentId)
`
