---
title: "getFirstMatchingEntry"
description: "Queries the first blocklist entry that matches an email address in Optimizely Campaign."
---

**Type**: String

**Parameters**

| Name         | Type   | Value                         |
| ------------ | ------ | ----------------------------- |
| sessionId    | String | ID of the session             |
| emailAddress | String | Email addresses to be queried |

**Return values**

First blocklist entry that matches the email address

**Code structure**

`String getFirstMatchingEntry(String sessionId, String emailAddress)
`
