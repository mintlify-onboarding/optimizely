---
title: "remove"
description: "Removes a recipient from the internal unsubscribe list currently unsubscribed."
---

If the recipient is contained in one of the recipient lists, he/she will receive mailings again.

**Type**: void

**Parameters**

| Name        | Type   | Value               |
| ----------- | ------ | ------------------- |
| sessionId   | String | ID of the session   |
| recipientId | String | ID of the recipient |

**Return values**

* **Code structure**  
`void remove(String sessionId, String recipientId)  
`
