---
title: "getColumnNames"
description: "Queries the column names of each blocklist entry in Optimizely Campaign."
---

**Type**: String\[ \]

**Parameters**

| Name      | Type   | Value             |
| --------- | ------ | ----------------- |
| sessionId | String | ID of the session |

**Return values**

Column names of the blocklist entries

The columns and the column order are subject to change.

**Code structure**

`String[] getColumnNames(String sessionId)
`
