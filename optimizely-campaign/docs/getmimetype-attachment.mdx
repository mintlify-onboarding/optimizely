---
title: "getMimeType"
description: "Queries the MIME type of an attachment in Optimizely Campaign."
---

**Type**: String

**Parameters**

| Name         | Type   | Value                |
| ------------ | ------ | -------------------- |
| sessionId    | String | ID of the session    |
| attachmentId | long   | ID of the attachment |

**Return values**

MIME type of the attachment

**Code structure**

`String getMimeType(String sessionId, long attachmentId)
`
