---
title: "Blocklist export job"
description: "Exports a blocklist into a CSV file on a regular basis."
---

* The blocklist export job exports a blocklist into a CSV file on a regular basis.
* Set the export frequency to daily, weekly or monthly.
* Export the complete list or only the changes since the last export.
* The file is compressed as a gzip file unless you demand otherwise.
