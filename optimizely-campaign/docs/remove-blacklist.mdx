---
title: "remove"
description: "Deletes an entry from the blocklist in Optimizely Campaign."
---

**Type**: void

**Parameters**

| Name      | Type   | Value                                                                                    |
| --------- | ------ | ---------------------------------------------------------------------------------------- |
| sessionId | String | ID of the session                                                                        |
| entry     | String | Entry (for example, an email address) to be deleted (can contain wildcards like ? or \*) |

**Return values**

* **Code structure**  
`void remove(String sessionId, String entry)  
`
