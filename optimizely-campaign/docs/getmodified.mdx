---
title: "getModified"
description: "Queries the date on which a coupon block was last modified in Optimizely Campaign."
---

**Type**: Date

**Parameters**

| Name      | Type   | Value                     |
| --------- | ------ | ------------------------- |
| sessionId | String | ID of the current session |
| blockId   | long   | ID of the coupon block    |

**Return values**

The last modification date of the coupon block

**Code structure**

`Date getModified(String sessionId, long blockId)
`
