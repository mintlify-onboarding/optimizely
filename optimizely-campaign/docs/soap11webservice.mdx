Methods defined within this web service are available through every web service. This web service cannot be accessed directly.

| Method                                             | Description                     |
| -------------------------------------------------- | ------------------------------- |
| [getVersion](/optimizely-campaign/docs/getversion) | Queries the version of the API. |
