---
title: "getRootFolders"
description: "Queries the IDs of the root folders of a specific folder type in Optimizely Campaign."
---

**Type**: long\[ \]

**Parameters**

| Name       | Type   | Value                     |
| ---------- | ------ | ------------------------- |
| sessionId  | String | ID of the current session |
| folderType | String | Type of the folder        |

**Return values**

IDs of the root folders

**Code structure**

`long[] getRootFolders(String sessionId, String folderType)
`
