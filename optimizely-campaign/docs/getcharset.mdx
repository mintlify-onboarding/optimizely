---
title: "getCharset"
description: "Queries the character set for the content (subject + body) of a mailing methods."
---

See also [setCharset](/optimizely-campaign/docs/setcharset).

**Type**: String

**Parameters**

| Name      | Type   | Value             |
| --------- | ------ | ----------------- |
| sessionId | String | ID of the session |
| mailingId | long   | ID of the mailing |

**Return values**

Character set for the content (subject + body) of the mailing

**Code structure**

`String getCharset(String sessionId, long mailingId)
`
