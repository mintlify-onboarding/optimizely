---
title: "setMimeType"
description: "Defines the MIME type of an attachment in Optimizely Campaign."
---

**Type**: void

**Parameters**

| Name         | Type   | Value                                                                                |
| ------------ | ------ | ------------------------------------------------------------------------------------ |
| sessionId    | String | ID of the session                                                                    |
| attachmentId | long   | ID of the attachment                                                                 |
| mimeType     | String | Corresponding MIME type of the fileExample: image/gif or text/comma-separated-values |

**Return values**

* **Code structure**  
`void setMimeType(String sessionId, long attachmentId, String mimeType)  
`
