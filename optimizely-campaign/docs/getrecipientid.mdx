---
title: "getRecipientId"
description: "Queries the ID of a recipient that received a mail in Optimizely Campaign."
---

**Type**: String

**Parameters**

| Name      | Type   | Value             |
| --------- | ------ | ----------------- |
| sessionId | String | ID of the session |
| mailId    | String | Mail ID           |

**Return values**

Recipient ID

**Code structure**

`String getRecipientId(String sessionId, String mailId)
`
