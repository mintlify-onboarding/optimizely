---
title: "Unsubscriber export job"
description: "Exports all unsubscribers into a CSV file on a regular basis."
---

* The unsubscriber export job exports all unsubscribers into a CSV file on a regular basis.
* Set the export frequency to daily, weekly or monthly.
* Export the complete list or only the changes since the last export.
* The file is compressed as a gzip file unless you demand otherwise.
