---
title: "getName"
description: "Queries the name of an opt-in process in Optimizely Campaign."
---

**Type**: String

**Parameters**

| Name           | Type   | Value                    |
| -------------- | ------ | ------------------------ |
| sessionId      | String | ID of the session        |
| optinProcessId | long   | ID of the opt-in process |

**Return values**

Name of the opt-in process

**Code structure**

`String getName(String sessionId, long optinProcessId)
`
