---
title: "getScheduleDate"
description: "Queries the sending date of a mailing in Optimizely Campaign."
---

**Type**: Date

**Parameters**

| Name      | Type   | Value             |
| --------- | ------ | ----------------- |
| sessionId | String | ID of the session |
| mailingId | long   | ID of the mailing |

**Return values**

Sending date of the mailing

**Code structure**

`Date getScheduleDate(String sessionId, long mailingId)
`
