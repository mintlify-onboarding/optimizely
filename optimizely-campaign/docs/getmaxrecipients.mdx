---
title: "getMaxRecipients"
description: "Queries the value of the Maximum number of recipients option in Optimizely Campaign."
---

**Type**: float

**Parameters**

| Name      | Type   | Value             |
| --------- | ------ | ----------------- |
| sessionId | String | ID of the session |
| mailingId | long   | ID of the mailing |

**Return values**

Number of recipients or the percentage of recipients that will receive the mailing after it has been started

The percentage is returned as a float from 0 to 1 (1 meaning 100%, see also: [isMaxRecipientsPercentage](/optimizely-campaign/docs/ismaxrecipientspercentage). The return value -1 means **All**. See also [setMaxRecipients](/optimizely-campaign/docs/setmaxrecipients).

**Code structure**

`float getMaxRecipients(String sessionId, long mailingId)
`
