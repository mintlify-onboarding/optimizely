---
title: "getIds"
description: "Queries the IDs of all available target groups in Optimizely Campaign. Temporary target groups are excluded."
---

**Type**: long\[ \]

**Parameters**

| Name      | Type   | Value             |
| --------- | ------ | ----------------- |
| sessionId | String | ID of the session |

**Return values**

IDs of all available target groups

**Code structure**

`long[] getIds(String sessionId)
`
