---
title: "uploadpersonalizedattachments"
description: "Uploads one or more attachments in Optimizely Campaign."
---

These attachments can be used when sending transactional mails. For more information regarding the Personalized Attachments feature, see [Personalized Attachments - Examples](/optimizely-campaign/docs/personalized-attachmentsexamples).

This operation is available in the form service.

> Tips
> 
> * The Personalized Attachment feature must first be activated by [customer support](/cdn-cgi/l/email-protection#0b787e7b7b64797f4b647b7f626662716e677225686466).
> * Personalized Attachments can only be used for event mailings.
> * Do not upload more than 500 kB at once.
> * With each call of this operation, up to five files can be uploaded.
> * An attachment is valid for 24 hours by default; i.e., after this time, the attachment can no longer be used for sending.
