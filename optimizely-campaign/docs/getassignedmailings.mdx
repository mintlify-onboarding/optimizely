---
title: "getAssignedMailings"
description: "Queries the IDs of the mailings in which a coupon block was used in Optimizely Campaign."
---

**Type**: long\[ \]

**Parameters**

| Name      | Type   | Value                     |
| --------- | ------ | ------------------------- |
| sessionId | String | ID of the current session |
| blockId   | long   | ID of the coupon block    |

**Return values**

The IDs of all mailings in which this block has been used

**Code structure**

`long[] getAssignedMailings(String sessionId, long blockId)
`
