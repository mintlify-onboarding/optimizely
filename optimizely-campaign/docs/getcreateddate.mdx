---
title: "getCreatedDate"
description: "Queries the creation date of a mailing in Optimizely Campaign."
---

**Type**: Date

**Parameters**

| Name      | Type   | Value             |
| --------- | ------ | ----------------- |
| sessionId | String | ID of the session |
| mailingId | long   | ID of the mailing |

**Return values**

Creation date of the mailing

**Code structure**

`Date getCreatedDate(String sessionId, long mailingId)
`
