---
title: "getRecipientListIds"
description: "Queries the IDs of the recipient lists of a mailing in Optimizely Campaign."
---

**Type**: long\[ \]

**Parameters**

| Name      | Type   | Value             |
| --------- | ------ | ----------------- |
| sessionId | String | ID of the session |
| mailingId | long   | ID of the mailing |

**Return values**

IDs of the assigned recipient lists of the mailing

**Code structure**

`long[] getRecipientListIds(string sessionId, long mailingId)
`
