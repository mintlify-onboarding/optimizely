---
title: "getColumnNames"
description: "Queries the column names of each mailing in Optimizely Campaign."
---

**Type**: String\[ \]

**Parameters**

| Name        | Type   | Value                                                                         |
| ----------- | ------ | ----------------------------------------------------------------------------- |
| sessionId   | String | ID of the session                                                             |
| mailingType | String | Type of the mailing (regular, event, confirmation, template, campaign, split) |

**Return values**

Column names of the mailings

The columns and the column order are subject to change.

**Code structure**

`String[] getColumnNames(String sessionId, String MailingType)
`
