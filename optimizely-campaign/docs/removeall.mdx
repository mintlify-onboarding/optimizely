---
title: "removeAll"
description: "Deletes multiple entries from the blocklist in Optimizely Campaign."
---

See also [remove](/optimizely-campaign/docs/remove-blacklist).

**Type**: void

**Parameters**

| Name      | Type        | Value                                                                                      |
| --------- | ----------- | ------------------------------------------------------------------------------------------ |
| sessionId | String      | ID of the session                                                                          |
| entries   | String\[ \] | Entries (for example, email addresses) to be deleted (can contain wildcards like ? or \*). |

**Return values**

* **Code structure**  
`void removeAll(String sessionId, String[] entries)  
`
