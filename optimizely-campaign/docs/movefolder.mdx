---
title: "moveFolder"
description: "Moves an existing folder into another existing folder in Optimizely Campaign."
---

**Type**: void

**Parameters**

| Name              | Type   | Value                        |
| ----------------- | ------ | ---------------------------- |
| sessionId         | String | ID of the current session    |
| folderId          | long   | ID of the folder to be moved |
| newParentFolderId | long   | ID of the new root folder    |

**Return values**: -

**Code structure**

`void moveFolder(String sessionId, long folderId, long newParentFolderId)
`
