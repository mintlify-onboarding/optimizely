---
title: "setMediaType"
description: "Defines a session's media type in Optimizely Campaign."
---

Some methods use the media type defined here.

**Type**: void

**Parameters**

| Name      | Type   | Value             |
| --------- | ------ | ----------------- |
| sessionId | String | ID of the session |
| mediaType | String | EMAIL or SMS      |

**Return values**

* **Code structure**  
`void setMediaType(String sessionId, String mediaType)  
`
