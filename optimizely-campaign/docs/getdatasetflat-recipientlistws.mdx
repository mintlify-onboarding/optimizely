---
title: "getDataSetFlat"
description: "Queries the values of each recipient list in Optimizely Campaign."
---

**Type**: String\[ \]

**Parameters**

| Name      | Type   | Value             |
| --------- | ------ | ----------------- |
| sessionId | String | ID of the session |

**Return values**

Values of the recipient lists

To assign the returned values to different recipient lists, use the [getColumnNames](/optimizely-campaign/docs/getcolumnnames-recipientlistws) method. With this method, you can determine the number of values of a recipient list.

**Code structure**

`String[] getDataSetFlat(String sessionId)
`
