---
title: "Multiple recipients export job"
description: "Exports multiple recipient lists into CSV files on a regular basis."
---

* The multiple recipients export job exports multiple recipient lists into CSV files on a regular basis.
* Set the export frequency to daily, weekly or monthly.
* Export the complete list or only the changes since the last export.
* The CSV files are named after the corresponding recipient list IDs (for example, 40291413820.csv) and are compressed as a gzip file unless you demand otherwise.
