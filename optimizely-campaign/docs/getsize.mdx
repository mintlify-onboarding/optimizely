---
title: "getSize"
description: "Queries the size of an attachment in Optimizely Campaign."
---

**Type**: long

**Parameters**

| Name         | Type   | Value                |
| ------------ | ------ | -------------------- |
| sessionId    | String | ID of the session    |
| attachmentId | long   | ID of the attachment |

**Return values**

Size of the attachment

**Code structure**

`long getSize(String sessionId, long attachmentId)
`
