---
title: "getChildren"
description: "Queries the IDs of all child folders of a folder in Optimizely Campaign."
---

**Type**: long\[ \]

**Parameters**

| Name      | Type   | Value                     |
| --------- | ------ | ------------------------- |
| sessionId | String | ID of the current session |
| folderId  | String | ID of the queried folder  |

**Return values**

IDs of all child folders

**Code structure**

`long[] getChildren(String sessionId, long folderId)
`
