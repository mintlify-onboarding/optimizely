**Parameters**

| Name          | Type      | Value                  |
| ------------- | --------- | ---------------------- |
| sessionId     | String    | ID of the session      |
| mailingId     | long      | ID of the mailing      |
| attachmentIds | long\[ \] | IDs of the attachments |

**Return values**

`void setAttachmentIds(String sessionId, long mailingId, long[] attachmentIds)
