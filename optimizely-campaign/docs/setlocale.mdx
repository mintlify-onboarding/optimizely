---
title: "setLocale"
description: "Defines the locale for a session in Optimizely Campaign."
---

**Type**: void

**Parameters**

| Name      | Type   | Value                                                                                                                                                          |
| --------- | ------ | -------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| sessionId | String | ID of the session                                                                                                                                              |
| locale    | String | A valid ISO language code (ISO-639)See: [http://www.loc.gov/standards/iso639-2/php/code\_list.php](http://www.loc.gov/standards/iso639-2/php/code%5Flist.php). |

**Return values**

* **Code structure**  
`void setLocale(String sessionId, String locale)  
`
