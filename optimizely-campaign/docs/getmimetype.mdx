---
title: "getMimeType"
description: "Queries the MIME type of a mailing in Optimizely Campaign."
---

**Type**: String

**Parameters**

| Name      | Type   | Value             |
| --------- | ------ | ----------------- |
| sessionId | String | ID of the session |
| mailingId | long   | ID of the mailing |

**Return values**

MIME type of the mailing

**Code structure**

`String getMimeType(String sessionId, long mailingId)
`
