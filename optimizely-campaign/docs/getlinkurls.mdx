---
title: "getLinkUrls"
description: "Queries the URLs of all tracking links that are used in a mailing in Optimizely Campaign."
---

**Type**: String\[ \]

**Parameters**

| Name      | Type   | Value             |
| --------- | ------ | ----------------- |
| sessionId | String | ID of the session |
| mailingId | long   | ID of the mailing |

**Return values**

URLs of all tracking links that are used in the mailing

**Code structure**

`String[] getLinkUrls(String sessionId, long mailingId)
`
