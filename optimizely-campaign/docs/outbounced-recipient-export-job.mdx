---
title: "Outbounced recipient export job"
description: "Exports recipients that exceeded the bounce limit into a CSV file on a regular basis."
---

* The outbounced recipient export job exports recipients that exceeded the bounce limit into a CSV file on a regular basis.
* Set the export frequency to daily, weekly or monthly.
* Export the complete list or only the changes since the last export.
* The file is compressed as a gzip file unless you demand otherwise.
