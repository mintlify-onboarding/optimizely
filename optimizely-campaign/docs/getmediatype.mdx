---
title: "getMediaType"
description: "Queries the media type of a session in Optimizely Campaign."
---

**Type**: String

**Parameters**

| Name      | Type   | Value             |
| --------- | ------ | ----------------- |
| sessionId | String | ID of the session |

**Return values**

Media type of the session: EMAIL or SMS

**Code structure**

`String getMediaType(String sessionId)
`
