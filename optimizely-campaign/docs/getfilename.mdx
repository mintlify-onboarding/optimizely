---
title: "getFilename"
description: "Queries the file name of an attachment in Optimizely Campaign."
---

**Type**: String

**Parameters**

| Name         | Type   | Value                |
| ------------ | ------ | -------------------- |
| sessionId    | String | ID of the session    |
| attachmentId | long   | ID of the attachment |

**Return values**

File name of the attachment

**Code structure**

`String getFilename(String sessionId, long attachmentId)
`
