---
title: "getDataSet"
description: "Queries the data of each attachment in Optimizely Campaign."
---

You can replace this method by the .NET replacement methods [getColumnNames](/optimizely-campaign/docs/getcolumnnames-attachment) and [getDataSetFlat](/optimizely-campaign/docs/getdatasetflat-attachment).

**Type**: String\[ \] \[ \]

**Parameters**

| Name      | Type   | Value             |
| --------- | ------ | ----------------- |
| sessionId | String | ID of the session |

**Return values**

A data set where the first row contains the column names and the following the values for each attachment.  
The columns and the column order of the data set are subject to change.

**Code structure**

`String[][] getDataSet(String sessionId)
`
