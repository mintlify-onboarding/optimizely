---
title: "getLocale"
description: "Queries the locale of a session in Optimizely Campaign."
---

**Type**: String

**Parameters**

| Name      | Type   | Value             |
| --------- | ------ | ----------------- |
| sessionId | String | ID of the session |

**Return values**

Valid ISO language code (ISO-639)

See: [http://www.loc.gov/standards/iso639-2/php/code\_list.php](http://www.loc.gov/standards/iso639-2/php/code%5Flist.php).

**Code structure**

`String getLocale(String sessionId)
`
