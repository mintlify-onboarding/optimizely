---
title: "getCodeCount"
description: "Counts the coupon codes contained in a coupon block in Optimizely Campaign."
---

**Type**: long

**Parameters**

| Name      | Type   | Value                     |
| --------- | ------ | ------------------------- |
| sessionId | String | ID of the current session |
| blockId   | long   | ID of the coupon block    |

**Return values**

The number of coupon codes in this block

**Code structure**

`long getCodeCount(String sessionId, long blockId)
`
