---
title: "getCurrentTime"
description: "Queries the current Optimizely Campaign server time in Optimizely Campaign."
---

This is useful for synchronizing the export of statistical response data.

**Type**: long

**Parameters**

| Name      | Type   | Value             |
| --------- | ------ | ----------------- |
| sessionId | String | ID of the session |

**Return values**

The current Optimizely Campaign server time in milliseconds

**Code structure**

`long getCurrentTime(String sessionId)
`
