---
title: "getCount"
description: "Counts all available target groups in Optimizely Campaign. Temporary target groups are excluded."
---

**Type**: int

**Parameters**

| Name      | Type   | Value             |
| --------- | ------ | ----------------- |
| sessionId | String | ID of the session |

**Return values**

Number of all available target groups

**Code structure**

`int getCount(String sessionId)
`
